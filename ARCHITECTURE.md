```

│  DashboardScreen                                                │

                 ↓ (Uses)
```

### **Incident Creation Flow**

```
┌─────────────────────────────────────────────────────────────┐
│  USER ACTION: Click "🚨 Medical Emergency"                  │
│  Location: dashboard_screen.dart                            │
│  Handler: _triggerIncident('Medical', 'Room 204')          │
└────────────────┬────────────────────────────────────────────┘
                 ↓
┌─────────────────────────────────────────────────────────────┐
│  FirebaseService.addIncident(type, location)                │
│                                                             │
│  1. Push new incident node                                 │
│     ref = _incidentsRef.push()                              │
│     id = ref.key (Firebase auto-ID)                         │
│                                                             │
│  2. Write incident data                                    │
│     await ref.set({                                         │
│       'type': 'Medical',                                    │
│       'location': 'Room 204',                               │
│       'status': 'active',                                   │
│       'timestamp': ISO8601_STRING,                          │
│       'aiReport': '⏳ Generating AI report...'              │
│     })                                                      │
│                                                             │
│  3. Trigger async AI report generation                     │
│     GeminiService.generateIncidentReport()                  │
│       .then((report) => updateAiReport(id, report))        │
└────────────────┬────────────────────────────────────────────┘
                 ↓                    ↓
        ┌────────────┐       ┌──────────────────┐
        │  FIREBASE  │       │  GEMINI API      │
        │  DATABASE  │       │  (Background)    │
        │  WRITES    │       │                  │
        │  INCIDENT  │       │  Processes       │
        │  DATA      │       │  Prompt &        │
        │            │       │  Generates       │
        │            │       │  AI Report       │
        │            │       │  (~3 seconds)    │
        └────────────┘       └────────┬─────────┘
                                     ↓
                        ┌──────────────────────┐
                        │  UpdateAiReport()    │
                        │                      │
                        │  await _incidentsRef │
                        │    .child(id)        │
                        │    .update({         │
                        │      'aiReport':...  │
                        │    })                │
                        └──────────┬───────────┘
                                   ↓
                        ┌──────────────────────┐
                        │  Firebase Event      │
                        │  Listener Fires      │
                        │  onValue event       │
                        └──────────┬───────────┘
                                   ↓
                        ┌──────────────────────┐
                        │  incidentsStream     │
```

## 🔌 API Integration

### **Gemini API Request Format**

```
POST /v1/models/gemini-2.0-flash:generateContent?key=API_KEY
Host: generativelanguage.googleapis.com
Content-Type: application/json

{
  "contents": [
    {
      "parts": [
        {
          "text": "You are an emergency response AI for SafeHaven hotel safety system.\nGenerate a brief incident report in under 60 words.\n\nIncident: Medical emergency at Room 204\nFormat exactly like this:\nSITUATION: [one sentence]\nACTION: [two immediate steps]\nPRIORITY: CRITICAL\nETA: 3 minutes"
        }
      ]
    }
  ],
  "generationConfig": {
    "temperature": 0.3,
    "maxOutputTokens": 120
  }
}

Response 200 OK:
{
  "candidates": [
    {
      "content": {
        "parts": [
          {
            "text": "SITUATION: Patient experiencing severe chest pain and shortness of breath.\nACTION: Call cardiology immediately. Prepare emergency resuscitation equipment.\nPRIORITY: CRITICAL\nETA: 3 minutes"
          }
        ]
      }
    }
  ]
}
```

### **Error Handling**

```
Response 401 Unauthorized:
├─ API key invalid/expired
├─ API key doesn't have permissions
└─ Solution: Regenerate key in Google Cloud Console

Response 404 Not Found:
├─ Model doesn't exist
├─ Wrong API version
└─ Solution: Use gemini-2.0-flash with /v1/ endpoint

Response 429 Rate Limited:
├─ Too many requests
├─ Quota exceeded
└─ Solution: Implement backoff retry logic

Response 500 Internal Server Error:
├─ Gemini API server issue
└─ Solution: Retry with exponential backoff
```

```
   ├─ Use Firebase Security Rules

   ├─ Use API key restrictions in Google Cloud
   └─ Rotate keys periodically
```


## 📱 UI State Management

### **StreamBuilder Pattern**

```dart
// LISTENING TO DATA CHANGES
StreamBuilder<List<Incident>>(
  stream: _firebaseService.incidentsStream,
  builder: (BuildContext context, AsyncSnapshot<List<Incident>> snapshot) {
    
    // WAITING STATE
    if (snapshot.connectionState == ConnectionState.waiting) {
      return Center(child: CircularProgressIndicator());
    }
    
    // ERROR STATE
    if (snapshot.hasError) {
      return Center(child: Text('Error: ${snapshot.error}'));
    }
    
    // DATA STATE (Update UI)
    final incidents = snapshot.data ?? [];
    return ListView(
      children: incidents.map((incident) => 
        IncidentCard(incident: incident)
      ).toList(),
    );
  },
)
```

**State Transitions:**
```
NONE → WAITING → ACTIVE (data arrives)
        ↓            ↓
      ERROR    Data Changes → ACTIVE (rebuild)
        ↓            ↓
      DONE    User navigates away → DONE (cleanup)
```

## ⚡ Performance Optimizations

### **Current Implementation**

```dart
// 1. Singleton pattern for services
static final FirebaseService _instance = FirebaseService._internal();
// → Only one database connection app-wide

// 2. Efficient sorting (on-demand)
incidents.sort((a, b) {
  try {
    final dateA = DateTime.parse(a.timestamp);
    final dateB = DateTime.parse(b.timestamp);
    return dateB.compareTo(dateA);  // Newest first
  } catch (e) { return 0; }
});
// → Sort happens once per update

// 3. Conditional rendering
if (incident.aiReport != null)
  AiReportCard(report: incident.aiReport!)
// → Only render if data exists

// 4. ListView.builder (not ListView)
ListView.builder(
  itemCount: incidents.length,
  itemBuilder: (context, index) => IncidentCard(...)
)
// → Only builds visible cards
```
